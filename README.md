Project Title: Zoo Management System (Python OOP)

Project Description:

This project is a Python-based Zoo Management System designed using Object-Oriented Programming (OOP) principles.
It manages animals, enclosures, staff, and health records while demonstrating clean code structure, inheritance, encapsulation, and unit testing.

What I Did:

Designed classes for animals, enclosures, staff, and health records
Applied OOP principles including inheritance, polymorphism, and encapsulation
Added data validation and implemented realistic system logic
Created a main script to demonstrate the system in action
Wrote unit tests for all major components
Used GitHub Classroom for version control and project submission

Skills Used:

Python, Object-Oriented Programming, Unit Testing, Software Design, Git, Debugging

Deliverables:

Python scripts, unit tests, and a working demonstration of the zoo system
